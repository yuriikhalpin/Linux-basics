```

```

```

Сохранение вывода в файл
```

cat /etc/hosts > /srv/3

cat /etc/shells > /srv/4
```

Сохранение вывода нескольких файлов в один файл
``` 
cat /etc/hosts >> /srv/merge

cat /etc/shells >> /srv/merge
```
